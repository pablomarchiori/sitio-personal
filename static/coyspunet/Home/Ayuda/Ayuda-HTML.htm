Compatibilidad con mensajes en formato HTML 

¿Puedo recibir mensajes HTML en mi programa de correo electrónico o sólo puedo recibir mensajes de texto sin formato? 
Algunos programas de correo electrónico tienen la capacidad de mostrar mensajes en formato HTML, el cual incluye gráficos y enlaces. Otros sólo pueden mostrar texto y/o gráficos limitados. Si intentás ver un mensaje HTML en un lector de correo electrónico de texto sin formato, el mensaje no se mostrará adecuadamente.

Los programas siguientes admiten HTML:

Microsoft Outlook 97 y 98 
Outlook Express 98 
Netscape 
Eudora Pro 
Correo electrónico de Web 
otros lectores de HTML

Los programas siguientes no admiten HTML:
Lotus Notes 
cc:Mail 
AOL 
otros lectores de texto sin formato

Si tenés alguna pregunta, consultá los archivos de la Ayuda de tu programa de correo electrónico en particular.
